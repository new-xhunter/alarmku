# Alarm Online Gratis

Alarm online gratis yang bisa diakses siapa saja dan di mana saja.

## Fitur
- Alarm dengan suara
- Responsive design
- Gratis tanpa biaya

## Cara Pakai
1. Buka [link website](https://new-xhunter.github.io/alarmku/)
2. Set waktu alarm
3. Klik set alarm

## Teknologi
- HTML
- CSS 
- JavaScript
